# short-track-roller-derby-scoreboard

A free, browser-based scoreboard for Short Track Roller Derby — built to work the way officials actually need it to work on game day.

Created by Officially Mel | YouTube: https://www.youtube.com/@melsguidetorollerderby

---------------------------------------------------------------------------

LIVE SCOREBOARD

https://officiallymel.github.io/short-track-roller-derby-scoreboard/short-track-scoreboard.html

No installation required — opens directly in any browser.

---------------------------------------------------------------------------

FEATURES

- Two periods of 10 jams each
- Period 1 runs clockwise; Period 2 runs counter-clockwise
- Period label displayed above JAM in center column
- Intermission overlay after Period 1 with current scores
- Unofficial Score overlay after Period 2 Jam 10 with live score editing
- Scores carry over between periods — combined running total throughout
- Undo last jam within current period
- Editable team names at game setup
- Keyboard shortcuts for fast operator input
- Color settings panel with 5 presets + custom color picker
- Hints toggle for a cleaner projection view
- Reset with confirm dialog — protected against accidental resets
- Fully offline — no internet required after loading

---------------------------------------------------------------------------

HOW TO USE

Option 1 — Use it instantly (no download)
Click the live link above. Opens directly in any browser — no install or account needed.

Option 2 — Download and run locally
1. Download short-track-scoreboard.html from this repository
2. Open it in Chrome, Edge, Firefox, or Safari
3. That's it — the file is fully self-contained

---------------------------------------------------------------------------

GAME DAY SETUP (PROJECTOR)

1. Plug in your projector
2. Set Windows display to Mirror (right-click desktop → Display Settings)
3. Open the HTML file in Chrome or Edge
4. Press F11 for full screen
5. Enter team names → Start Game
6. Press H to hide keyboard hints before the bout begins

---------------------------------------------------------------------------

KEYBOARD SHORTCUTS

Q         Team 1 +1
A         Team 1 -1
P         Team 2 +1
L         Team 2 -1
Space     End Jam / Advance
H         Toggle keyboard hint visibility
Esc       Close dialogs

---------------------------------------------------------------------------

END OF GAME FLOW

1. Jam 10 ends → Unofficial Score screen appears with live score editing
2. Adjust any final score corrections using the +1 / -1 buttons
3. Click Confirm — Show Final Score
4. Final Score is locked
5. Use Reset to start a new game
